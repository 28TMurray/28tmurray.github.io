tierlist.py ↗

Tierlist

The second Discord bot, and it had by far had the longest run. I am constantly amazed how this one managed to work consistently given how dense with awful decisions it is. A random message will get reacted with S-F by the bot, and everytime someone reacts it writes out to a csv file containing a lot of info including how many of each reaction the message has.

Commands:

  • !exempt, !unexempt - the bot won't tierlist your message and vice-versa.
  • !rm - if you reply to your own message with !rm, the bot removes the reactions and removes it from the csv file.
  • !histogram - written by a friend. Sends an image of a histogram. Use on its own for all messages, or reply to a specific message to see the histogram for the one message.
anyboard.py ↗

Anyboard

The best of my Discord bots. This one works like a normal starboard, but it automatically creates new boards for any emoji, including custom, with the board messages all having nice embeds and handling for attachments. Uses a json file to store and load.

Commands:

  • !exempt, !unexempt - the bot won't board your message and vice-versa.
  • !rm - if you reply to your own message with !rm, the bot deletes it from any boards it is in and removes it from the json.
